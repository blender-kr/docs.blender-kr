# 블렌더 한국어 문서

## 기여 방법

이 프로젝트에 기여하려면 다음 단계를 따라주세요:

1. 이 저장소를 포크합니다.
2. 로컬 환경에 저장소를 클론합니다:

    ```bash
    git clone https://github.com/your-username/docs.blender-kr.git
    ```

3. 새로운 브랜치를 생성합니다:

    ```bash
    git checkout -b feature/your-feature-name
    ```

4. 필요한 변경 사항을 적용하고 커밋합니다:

    ```bash
    git commit -m "docs: 변경 사항에 대한 간단한 설명"
    ```

5. 원격 저장소에 브랜치를 푸시합니다:

    ```bash
    git push origin feature/your-feature-name
    ```

6. GitHub에서 풀 리퀘스트를 생성합니다.

기여해 주셔서 감사합니다!
